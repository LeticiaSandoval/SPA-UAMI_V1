Guía rápida de instalción de SPA-UAMI_V1
1.- Copiar en xampp/htdocs la carpeta SPA-UAMI_V1
2.- Importar la BD (productividad_academica.sql) en phpMyAdmin (ver anexo para detalles)
    En breve: desde la interfaz de xampp, selecionar boton admin (MySQL) y boton admin (Apache)
	importar la BD productividad_academica.sql (dejar todas las opciones por defecto)
3.- Abrir  en un navegador: http://localhost/SPA-UAMI_V1
4.- Iniciar sesión con los usuarios que se encuentran dentro de la base de datos
